Program State‚Äã


The program state governs the Switchboard V2 program and controls the token mint used throughout for oracle rewards, aggregator leases, and other incentives.


The Switchboard V2 program can support many oracle queue's, each acting as independent networks with their own oracles, configuration, and security model.
